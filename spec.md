# 코인 모니터링 웹 서비스 기능 명세서

## 1. 개요

사용자가 원하는 암호화폐를 선택하여 목록을 만들고, 해당 코인의 실시간 가격과 거래량을 추적하며, 지정한 가격에 도달했을 때 알림을 받을 수 있는 간소화된 코인 모니터링 웹 서비스입니다.

## 2. 핵심 기능

### 2.1. 관심 코인 관리

- **설명:** 사용자가 자신만의 코인 목록을 만들고 관리할 수 있습니다.
- **주요 액션:**
  - 사용자는 암호화폐를 검색할 수 있습니다.
  - 검색 결과에서 원하는 코인을 '관심 목록'에 추가할 수 있습니다.
  - 관심 목록에 추가된 코인을 제거할 수 있습니다.
- **화면 구성:**
  - 코인 검색을 위한 입력창.
  - 검색된 코인을 추가하는 '추가' 버튼.
  - 목록의 각 코인 항목을 삭제하는 '삭제' 버튼.

### 2.2. 실시간 시세 및 거래량 조회

- **설명:** 관심 목록에 추가된 코인들의 현재가와 24시간 거래량을 실시간으로 표시합니다.
- **주요 액션:**
  - 사용자는 관심 목록에 있는 코인들의 정보를 한눈에 볼 수 있습니다.
  - 데이터는 주기적으로 자동 갱신되어야 합니다. (예: 30초마다)
- **표시 정보 (목록의 각 코인별):**
  - 코인 이름 (예: Bitcoin)
  - 코인 심볼 (예: BTC)
  - 현재 가격 (예: ₩90,000,000)
  - 24시간 거래량

### 2.3. 지정 가격 알림

- **설명:** 사용자가 관심 목록의 특정 코인에 대해 목표 가격을 설정하고, 해당 가격에 도달하면 알림을 받습니다.
- **주요 액션:**
  - 사용자는 목록에서 코인을 선택하여 알림을 설정할 수 있습니다.
  - 목표 가격(이상/이하)을 입력할 수 있습니다. (예: 비트코인이 ₩95,000,000 이상이 되면 알림)
  - 설정된 알림을 확인하고 삭제할 수 있습니다.
- **알림 방식:**
  - 브라우저 푸시 알림을 통해 사용자에게 즉시 전달됩니다.
- **화면 구성:**
  - 각 코인 항목 옆에 '알림 설정' 아이콘 (예: 종 모양).
  - 알림 설정 아이콘 클릭 시 목표 가격을 입력할 수 있는 간단한 폼(모달 창 등).
